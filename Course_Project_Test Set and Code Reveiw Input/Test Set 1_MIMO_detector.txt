In order for you to match the answers for the MIMO detector test case, 
please use the following definition of the QPSK constellation:

(1+j) corresponds to bit1=0  bit2=0 
(-1+j) corresponds to 0 1
(-1-j) corresponds to 1 1
(1-j) corresponds to 1 0

This means that we set Es=2 (energy per symbol), and the noise level should be 
calculated according to Equation (31) in Hochwald's paper 
"Achieving near-capacity on a multiple-antenna channel". 

Consider one transmission from two TX/RX antennas corresponding to
four bits (two QPSK symbols): b0, b1, b2, b3.
Assume that
Eb/N0=2db (following the definition given in the paper)

Mc=2 #QPSK
Mt=2
Nr=2
Code_R=1/2  # Code Rate

Es=4 # in linear
## BE CAREFUL ##
#Es here is total energy(not average energy per symbol) through Tx antenna
#Es = (energy per symbol * Mt) = 2(=energy per symbol) * 2 (Tx antennas)

EbN0=2 #in dB

sigma2=(Es/2)*(Nr/(Code_R*Mt*Mc))*(10**(-EbN0/10))
